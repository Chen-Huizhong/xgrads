Welcome to xgrads's documentation!
==================================

.. image:: https://raw.githubusercontent.com/miniufo/xgrads/master/pics/3D.png

This is a Python package to parse and read binary dataset described by a
`.ctl <http://cola.gmu.edu/grads/gadoc/descriptorfile.html>`__ file commonly
used by `GrADS <http://cola.gmu.edu/grads/>`__ or
`openGrADS <http://www.opengrads.org/>`__.  This project is
`published on GitHub <https://github.com/miniufo/xgrads>`__ and can be cited
using its `Zenodo DOI <https://doi.org/10.5281/zenodo.6526520>`__.
